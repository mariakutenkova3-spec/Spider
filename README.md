# Spider
Библиотеки для управления квадроподом, Arduino IDE 
Квадропод - робот-паук на 4 лапах, на каждой лапе по 2 сервопривода. Оснащен пищалкой, ультразвуковым датчиком и двумя RGB-светодиодами. 
В библиотеках есть как простейшие движения/действия, управлящие вышеназванными элементами через Arduino Nano, так и сложные "параллельные"
движения. В лабораторных работах описано подключение и кодирование всех элементов.
